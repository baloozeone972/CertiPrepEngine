# Microsoft Azure Developer Associate (AZ-204)

## 📋 Description
La certification Microsoft Azure Developer Associate (AZ-204) est une certification pratique qui évalue les compétences nécessaires pour développer des solutions cloud sur Microsoft Azure. Cette certification couvre une large gamme de thèmes essentiels pour les développeurs Azure, y compris le développement d'applications sur App Services, l'utilisation de Functions et Storage, la gestion des bases de données SQL, ainsi que les aspects d'authentification, sécurité, surveillance et journalisation.

## 🏛️ Organisme délivrant
**Microsoft Corporation** - Leader mondial dans le domaine de la technologie, Microsoft fournit des solutions cloud innovantes et des certifications reconnues pour les professionnels de l'informatique.

## 📅 Validité
- **Valable 2 ans**
- Renouvellement par examen ou certification supérieure (Azure DevOps Engineer Expert, Azure Solutions Architect Expert)

## 📚 Programme détaillé
### Thèmes principaux
| Thème | Questions |
|---|---|
| Azure Compute (App Services) | 60 |
| Azure Functions | 60 |
| Azure Storage | 60 |
| Azure SQL | 50 |
| Authentication & Security | 60 |
| Monitoring & Logging | 60 |

### Compétences évaluées
- Développer et déployer des applications sur Azure App Services
- Créer, configurer et gérer des fonctions Azure
- Utiliser les services de stockage Azure pour stocker et accéder aux données
- Créer, configurer et gérer des bases de données SQL Azure
- Sécuriser les applications et les services Azure
- Surveiller et gérer les performances des applications sur Azure
