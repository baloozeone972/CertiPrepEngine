# Certified Kubernetes Administrator (CKA)

## 📋 Description
La certification CKA (Certified Kubernetes Administrator) est une certification **pratique** qui évalue les compétences techniques d'un administrateur Kubernetes capable de gérer efficacement des clusters complexes. Cette certification est délivrée par la Cloud Native Computing Foundation (CNCF), une fondation open source sous la Linux Foundation, et est reconnue dans le monde entier pour son niveau élevé de difficulté.

## 🏛️ Organisme délivrant
**Cloud Native Computing Foundation (CNCF)** - Fondation open source sous la Linux Foundation

## 📅 Validité
- **Valable 2 ans**
- Renouvellement par examen ou certification supérieure (CKS)

## 📚 Programme détaillé
### Thèmes principaux
| Thème | Questions |
|---|---|
| Cluster Architecture | 40 |
| Workloads & Scheduling | 50 |
| Services & Networking | 50 |
| Storage | 40 |
| Troubleshooting | 50 |
| Security | 40 |
| Helm & Operators | 30 |

### Compétences évaluées
- Installer et configurer un cluster Kubernetes
- Gérer des workloads et des schédulements complexes
- Configurer des services et réseaux Kubernetes
- Gérer les stockages dans un environnement Kubernetes
- Dépanner des problèmes courants sur les clusters Kubernetes
- Sécuriser un cluster Kubernetes
- Utiliser Helm et des opérateurs pour gérer les applications
