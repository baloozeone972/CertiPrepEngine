# Chef Developer

## 📋 Description
La certification **Chef Developer** est une certification professionnelle qui évalue les compétences en Infrastructure as Code avec Chef, un outil puissant et flexible pour la gestion des infrastructures. Cette certification offre une approche orientée sur les ressources, les recettes et les cookbooks, permettant aux professionnels de gérer efficacement leur infrastructure en code.

## 🏛️ Organisme délivrant
**The Chef Software, Inc.** - Une entreprise leader dans le domaine des outils de gestion d'infrastructure en code.

## 📅 Validité
- **Valable 2 ans**
- Renouvellement par examen ou certification supérieure.

## 📚 Programme détaillé
### Thèmes principaux
| Thème | Questions |
|---|---|
| Chef Resources | 60 |
| Recipes & Cookbooks | 70 |
| Chef Workstation | 40 |
| Chef Server | 40 |
| Inspec | 40 |

### Compétences évaluées
- Gestion des ressources Chef (nodes, roles, environments)
- Création et gestion de recettes et cookbooks
- Utilisation du Workstation pour le développement et la test
- Configuration et gestion d'un serveur Chef
- Utilisation de l'outil Inspec pour la vérification et le contrôle qualité
