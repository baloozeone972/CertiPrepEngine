# Docker Certified Associate (DCA)

## 📋 Description
La certification Docker Certified Associate (DCA) est une certification pratique qui évalue les compétences nécessaires pour gérer des conteneurs et des applications avec Docker. Elle couvre les bases de la containerisation, la création d'images et de fichiers Dockerfile, le networking Docker, les volumes et le stockage, ainsi que la gestion de services avec Docker Compose et Swarm. Cette certification est essentielle pour les professionnels qui souhaitent se spécialiser dans le domaine de la conteneurisation et de l'orchestration avec Docker.

## 🏛️ Organisme délivrant
**Docker, Inc.** - Développeur et éditeur de logiciels open source

## 📅 Validité
- **Valable 2 ans**
- Renouvellement par examen ou certification supérieure (DCK)

## 📚 Programme détaillé
### Thèmes principaux
| Thème | Questions |
|---|---|
| Docker Basics | 60 |
| Images et Dockerfiles | 70 |
| Networking Docker | 50 |
| Volumes et Stockage | 50 |
| Docker Compose | 60 |
| Docker Swarm | 60 |

### Compétences évaluées
- Gérer les conteneurs avec Docker CLI
- Créer et gérer des images Docker
- Utiliser les fichiers Dockerfile pour automatiser la création d'images
- Configurer le networking Docker
- Utiliser les volumes et le stockage pour persister des données
- Déployer et gérer des applications avec Docker Compose
- Orchestrer des services avec Docker Swarm
