# Puppet Professional

## 📋 Description
La certification Puppet - Gestion de configuration et infrastructure as code avec Puppet est une certification **pratique** qui évalue votre capacité à gérer des infrastructures complexes en utilisant Puppet, un outil de gestion de configuration open source. Cette certification couvre les aspects clés du langage DSL Puppet, la création et l'utilisation de modules, le développement de classes et de types définis, l'utilisation de Hiera pour la gestion des données et le fonctionnement du serveur Puppet.

## 🏛️ Organisme délivrant
**Puppet, Inc.** - Leader mondial dans la gestion de configuration et l'infrastructure as code

## 📅 Validité
- **Valable 3 ans**
- Renouvellement par examen ou certification supérieure

## 📚 Programme détaillé
### Thèmes principaux
| Thème | Questions |
|---|---|
| Puppet DSL | 70 |
| Modules | 60 |
| Classes & Defined Types | 60 |
| Hiera | 50 |
| Puppet Server | 60 |

### Compétences évaluées
- Créer et utiliser des ressources personnalisées en Puppet DSL
- Développer et gérer des modules Puppet efficacement
- Utiliser les classes et les types définis pour organiser la configuration
- Gérer les données de configuration avec Hiera
- Configurer et gérer un serveur Puppet pour une gestion d'infrastructure
