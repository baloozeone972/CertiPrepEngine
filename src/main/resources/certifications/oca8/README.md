# Oracle Certified Associate (OCA) - Java 8

## 📋 Description

La certification Oracle Certified Associate (OCA) Java 8 est la certification de niveau **débutant** qui valide les
compétences fondamentales en programmation Java. Elle est conçue pour les développeurs ayant une expérience initiale
avec Java.

## 🏛️ Organisme délivrant

**Oracle University** - Division certification d'Oracle Corporation

## 📅 Validité

- **Valable à vie** (non expirable)
- Java 8 reste largement utilisé, mais Oracle recommande de passer à Java 11/17 pour rester à jour

## 📚 Programme détaillé

### Thèmes principaux

| Thème                           | Pourcentage |
|---------------------------------|-------------|
| Fondamentaux Java               | 15%         |
| Types de données et opérateurs  | 15%         |
| Structures de contrôle          | 10%         |
| POO - Héritage et polymorphisme | 20%         |
| Exceptions                      | 10%         |
| API de base (String, ArrayList) | 15%         |
| Lambda et Streams               | 10%         |
| Date et Time API                | 5%          |

### Compétences évaluées

- Créer et exécuter des programmes Java
- Utiliser les types primitifs et wrappers
- Gérer les opérateurs et les conversions
- Maîtriser les structures conditionnelles et boucles
- Comprendre l'héritage et le polymorphisme
- Gérer les exceptions (try-catch-finally)
- Utiliser les collections (ArrayList, HashMap)
- Programmer avec l'API Date/Time (Java 8+)

⭐⭐⭐ (Élevé - Certification standard de l'industrie)

## 🎯 Objectif

Valider les compétences de base en développement Java pour débuter une carrière de développeur Java.

## 🚀 Débouchés

- Développeur Java Junior
- Développeur Full Stack Java
- Intégrateur technique
- Support technique Java

## 📊 Statistiques

- **Taux de réussite** : ~65-70%
- **Temps de préparation recommandé** : 80-120 heures
- **Format examen** : QCM, 80 questions, 150 minutes
