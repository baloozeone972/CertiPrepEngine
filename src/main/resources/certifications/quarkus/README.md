## 📋 Description

Quarkus est un framework Java Kubernetes Native conçu pour les environnements cloud et les architectures serverless. La certification **Quarkus Professional Red Hat** atteste de votre capacité à développer des microservices réactifs, à optimiser la consommation mémoire, à utiliser les extensions Quarkus (RESTEasy Reactive, Hibernate with Panache, Kafka) et à produire des exécutables natifs via GraalVM.

Cette certification est idéale pour les développeurs Java souhaitant se spécialiser dans les applications cloud‑natives à faible empreinte mémoire et à démarrage rapide. Elle est particulièrement reconnue chez les partenaires et clients Red Hat utilisant OpenShift.

## 🏛️ Organisme délivrant

**Red Hat, Inc.** – L’examen est administré par Red Hat (EX371 ou équivalent) et donne droit au titre **Red Hat Certified Specialist in Quarkus**.

## 📅 Validité

La certification est **valide 3 ans**. Le renouvellement peut s’effectuer en repassant l’examen ou en obtenant une certification de niveau supérieur (ex. Red Hat Certified Architect).

## 📚 Programme détaillé

| Thème | Compétences évaluées |
|-------|----------------------|
| **Principes fondamentaux de Quarkus** | Architecture, cycle de vie, configuration, profils, injection de dépendances (CDI), extensions. |
| **Développement d'applications REST** | RESTEasy Reactive, validation, sérialisation JSON, documentation OpenAPI, clients REST. |
| **Persistance des données** | Hibernate ORM with Panache, transactions, accès aux bases de données relationnelles et NoSQL (MongoDB avec Panache). |
| **Messagerie réactive** | Reactive Messaging, intégration avec Apache Kafka, AMQP, modèles réactifs (Mutiny). |
| **Compilation native et déploiement** | Génération d'exécutables natifs avec GraalVM, conteneurisation, déploiement sur Kubernetes/OpenShift, health checks, métriques. |
