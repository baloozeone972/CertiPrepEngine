# AWS Certified Developer - Associate (DVA-C02)

## 📋 Description
La certification AWS Certified Developer - Associate (DVA-C02) est une certification **pratique** qui évalue les compétences nécessaires pour développer et déployer des applications cloud-native sur AWS. Elle couvre une large gamme de services, notamment Lambda, API Gateway, DynamoDB et S3. Cette certification est idéale pour les développeurs souhaitant s'adapter aux technologies AWS et développer des applications performantes sur la plateforme cloud.

## 🏛️ Organisme délivrant
**Amazon Web Services (AWS)** - Leader mondial du cloud computing

## 📅 Validité
- **Valable 3 ans**
- Renouvellement possible par examen ou certification supérieure

## 📚 Programme détaillé
### Thèmes principaux
| Thème | Questions |
|---|---|
| Lambda & Serverless | 80 |
| API Gateway | 60 |
| DynamoDB | 70 |
| S3 & Storage | 60 |
| CI/CD (CodeSuite) | 70 |
| Security & IAM | 60 |

### Compétences évaluées
- Conception et mise en œuvre de fonctions Lambda
- Utilisation d'API Gateway pour exposer des services web
- Création et gestion de bases de données DynamoDB
- Gestion des stockages S3
- Intégration continue et livraison continue avec CodePipeline et CodeBuild
- Sécurité des applications et gestion des identités et des accès
