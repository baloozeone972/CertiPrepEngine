# Ansible Automation (Red Hat EX294)

## 📋 Description
La certification Ansible Automation est une certification **pratique** qui évalue votre capacité à automatiser des infrastructures et des applications en utilisant le logiciel Ansible. Elle couvre les aspects fondamentaux de l'automatisation avec Ansible, allant des concepts de base aux fonctionnalités avancées.

## 🏛️ Organisme délivrant
**Red Hat** - Leader mondial dans la création et la distribution de logiciels, notamment des solutions pour les systèmes d'exploitation, le cloud et la sécurité.

## 📅 Validité
- **Valable 2 ans**
- Renouvellement par examen ou certification supérieure (pourrait être une autre certification Ansible)

## 📚 Programme détaillé
### Thèmes principaux
| Thème | Questions |
|---|---|
| Ansible Core | 60 |
| Playbooks | 70 |
| Roles | 60 |
| Variables & Facts | 60 |
| Templates (Jinja2) | 50 |

### Compétences évaluées
- Installer et configurer Ansible sur un système d'exploitation
- Créer des playbooks pour automatiser des tâches
- Utiliser les rôles pour organiser et réutiliser le code Ansible
- Manipuler les variables et les faits dans Ansible
- Utiliser les templates Jinja2 pour générer des fichiers de configuration
