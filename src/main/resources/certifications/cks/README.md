# Certified Kubernetes Security Specialist (CKS)

## 📋 Description
La certification CKS (Certified Kubernetes Security Specialist) est une certification **pratique** qui évalue les compétences de sécurité avancées des clusters Kubernetes. Elle couvre un large éventail de thèmes essentiels pour la sécurité des systèmes et applications basés sur Kubernetes, garantissant que les administrateurs et spécialistes de sécurité sont prêts à gérer les défis de sécurité dans un environnement Kubernetes.

## 🏛️ Organisme délivrant
**Cloud Native Computing Foundation (CNCF)** - Fondation open source sous la Linux Foundation

## 📅 Validité
- **Valable 2 ans**
- Renouvellement par examen ou certification supérieure (CKS)

## 📚 Programme détaillé
### Thèmes principaux
| Thème | Questions |
|---|---|
| Cluster Hardening | 30 |
| System Hardening | 30 |
| Supply Chain Security | 30 |
| Monitoring & Logging | 30 |
| Runtime Security | 30 |
| Compliance | 50 |

### Compétences évaluées
- Configurer et sécuriser un cluster Kubernetes
- Mettre en œuvre des pratiques de sécurité système avancées
- Assurer la sécurité dans le processus d'approvisionnement des systèmes et applications (supply chain)
- Configurer et utiliser les outils de surveillance et de journalisation pour détecter et corriger les vulnérabilités
- Sécuriser le runtime Kubernetes en appliquant des bonnes pratiques et en utilisant les outils appropriés
- Assurer la conformité des systèmes Kubernetes aux réglementations et normes applicables
