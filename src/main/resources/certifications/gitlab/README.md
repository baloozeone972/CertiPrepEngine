# GitLab Associate

## 📋 Description
La certification GitLab Associate est une certification **pratique** qui évalue votre maîtrise des principes de CI/CD et DevOps avec GitLab. Cette certification vous permet d'acquérir des compétences essentielles pour gérer et automatiser les pipelines de développement, la mise en production et la sécurité des applications avec GitLab.

## 🏛️ Organisme délivrant
**GitLab Inc.** - Une entreprise leader dans le domaine des outils de gestion de la version et du développement logiciel.

## 📅 Validité
- **Valable 2 ans**
- Renouvellement par examen ou certification supérieure (GitLab Developer)

## 📚 Programme détaillé
### Thèmes principaux
| Thème | Questions |
|---|---|
| GitLab CI/CD | 80 |
| GitLab Runner | 50 |
| Git Basics | 50 |
| GitLab Security | 40 |
| GitLab Packages | 30 |

### Compétences évaluées
- Configurer et utiliser GitLab CI/CD pour automatiser les pipelines de développement
- Gérer des runners pour exécuter les jobs du pipeline
- Maîtriser les bases de Git et la gestion des dépôts
- Assurer la sécurité des projets avec GitLab Security
- Utiliser les fonctionnalités de gestion des paquets de GitLab
