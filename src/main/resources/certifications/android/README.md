# Android Developer (Google)

## 📋 Description

Certification Android - Développement d'applications mobiles Android avec Kotlin/Java, UI/UX, et architecture moderne.

## 🏛️ Organisme délivrant

**Google**

## 📅 Validité

- **Valable 2 ans**
- Renouvellement par examen

## 📚 Programme

- Activities et Fragments
- UI/UX Design (Jetpack Compose)
- Data Storage (Room, DataStore)
- Networking (Retrofit)
- Architecture (MVVM, ViewModel)
- Security (Keystore, Biometric)

⭐⭐⭐⭐ (Élevé - Standard pour développeurs Android)

- Confirmé : 50 000 - 70 000 €
- Senior : 70 000 - 90 000 €
