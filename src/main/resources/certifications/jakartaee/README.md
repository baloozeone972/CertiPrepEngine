# Jakarta EE Developer (Eclipse Foundation)

## 📋 Description
La certification Jakarta EE Developer est une certification **théorique et pratique** qui évalue les compétences nécessaires pour développer des applications d'entreprise utilisant le framework Jakarta EE (ex-Java EE). Cette certification couvre les principaux aspects de la plateforme Jakarta EE, offrant une solide base pour les professionnels souhaitant s'engager dans le développement d'applications modernes et sécurisées.

## 🏛️ Organisme délivrant
**Eclipse Foundation** - Fondation open source axée sur le développement et la diffusion de technologies open source, dont Jakarta EE est une partie importante.

## 📅 Validité
- **Valable 3 ans**
- Renouvellement possible par examen ou certification supérieure

## 📚 Programme détaillé
### Thèmes principaux
| Thème | Questions |
|---|---|
| Jakarta Servlet | 60 |
| Jakarta Server Pages (JSP) | 50 |
| Jakarta Persistence (JPA) | 70 |
| Jakarta Enterprise Beans (EJB) | 60 |
| Jakarta REST (JAX-RS) | 50 |
| Jakarta Security | 60 |

### Compétences évaluées
- Conception et développement d'applications Web avec Jakarta Servlet et JSP
- Gestion de la persistance des données avec Jakarta Persistence (JPA)
- Développement d'applications distribuées et sécurisées avec Jakarta Enterprise Beans (EJB) et Jakarta Security
- Création d'API RESTful avec Jakarta REST (JAX-RS)
