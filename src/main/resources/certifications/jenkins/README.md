# Jenkins Engineer (CloudBees)

## 📋 Description

La certification Jenkins Engineer valide les compétences en CI/CD avec Jenkins, incluant les pipelines, les plugins, les
shared libraries, et la gestion master/agent.

## 🏛️ Organisme délivrant

**CloudBees** - Leader de l'automatisation CI/CD (mainteneur de Jenkins)

## 📅 Validité

- **Valable 2 ans**
- Renouvellement par examen

## 📚 Programme détaillé

### Thèmes principaux

| Thème                | Pourcentage |
|----------------------|-------------|
| Jenkins Architecture | 20%         |
| Pipeline as Code     | 30%         |
| Plugins              | 15%         |
| Shared Libraries     | 15%         |
| Security             | 10%         |
| Master/Agent         | 10%         |

### Compétences évaluées

- Installer et configurer Jenkins
- Créer des pipelines déclaratifs et scriptés
- Gérer les plugins et leurs dépendances
- Développer des shared libraries
- Sécuriser Jenkins (RBAC, credentials)
- Gérer les agents (master/agent)
