# Microsoft Azure Fundamentals (AZ-900)

## 📋 Description
La certification **Microsoft Azure Fundamentals (AZ-900)** est une certification **conçue pour les débutants** dans le domaine du cloud Microsoft Azure. Elle vise à fournir une compréhension solide des concepts de base du cloud, des services Azure essentiels et des meilleures pratiques pour la sécurité et le support. Cette certification est idéale pour les professionnels qui souhaitent démarrer leur carrière dans le cloud Microsoft Azure et acquérir les compétences nécessaires pour réussir dans des rôles tels que Cloud Administrator, DevOps Engineer, et Azure Solutions Architect.

## 🏛️ Organisme délivrant
**Microsoft Corporation** - Leader mondial du cloud et technologie

## 📅 Validité
- **Valable 2 ans**
- Renouvellement par examen ou certification supérieure

## 📚 Programme détaillé
### Thèmes principaux
| Thème | Questions |
|---|---|
| Cloud Concepts | 50 |
| Azure Core Services | 80 |
| Security & Compliance | 60 |
| Pricing & Support | 60 |
| Azure Management Tools | 50 |

### Compétences évaluées
- Comprendre les concepts de base du cloud et la façon dont ils s'appliquent à Azure
- Identifier les services Azure essentiels et comprendre leurs utilités
- Configurer la sécurité et la conformité dans Azure
- Comprendre les modèles de tarification et le support d'Azure
- Utiliser des outils de gestion Azure pour la création et la supervision des ressources
