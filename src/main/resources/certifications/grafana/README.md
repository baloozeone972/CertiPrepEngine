## 📋 Description

Grafana est la plateforme open source leader pour la visualisation et l'analyse des métriques, logs et traces. La certification **Grafana Associate** atteste de votre capacité à concevoir des dashboards interactifs, à connecter diverses sources de données (Prometheus, Loki, MySQL, etc.), à configurer des alertes intelligentes et à gérer les utilisateurs et les permissions dans une instance Grafana.

Elle s’adresse aux ingénieurs DevOps, SRE, administrateurs systèmes et analystes de données qui souhaitent valoriser leur expertise en observabilité.

## 🏛️ Organisme délivrant

**Grafana Labs** – L’examen est administré en ligne via la plateforme de certification Grafana.

## 📅 Validité

La certification est **valide 2 ans**. Le renouvellement s’effectue en repassant l’examen (une version actualisée est disponible).

## 📚 Programme détaillé

| Thème | Compétences évaluées |
|-------|----------------------|
| **Fondamentaux de Grafana** | Architecture, installation, configuration, concepts clés (organisation, dossiers, tableaux de bord). |
| **Dashboards et visualisations** | Création de dashboards, types de panneaux (graph, stat, table, heatmap), variables, transformations de données, liens. |
| **Sources de données** | Configuration des sources (Prometheus, Loki, Elasticsearch, MySQL, Postgres), requêtes, modèles de données. |
| **Alertes** | Création de règles d'alerte, gestion des notifications (canaux), étiquettes, silences. |
| **Administration et sécurité** | Gestion des utilisateurs, équipes, rôles, permissions, authentification (OAuth, LDAP), provisioning. |
