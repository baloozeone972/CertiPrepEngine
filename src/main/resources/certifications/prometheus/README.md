# Prometheus Associate (CNCF)

## 📋 Description
La certification Prometheus Associate est une formation et un examen **pratique** qui évalue votre compréhension et vos compétences en matière de monitoring et d'observabilité avec Prometheus. Cette certification est délivrée par la Cloud Native Computing Foundation (CNCF), une organisation à but non lucratif qui promeut le développement et l'adoption de technologies open source pour la cloud computing.

## 🏛️ Organisme délivrant
**Cloud Native Computing Foundation (CNCF)** - Fondation open source sous la Linux Foundation

## 📅 Validité
- **Valable 2 ans**
- Renouvellement par examen ou certification supérieure (Prometheus Professional)

## 📚 Programme détaillé
### Thèmes principaux
| Thème | Pourcentage |
|---|---|
| Prometheus Architecture | 20% |
| Metrics and Exporters | 30% |
| PromQL | 40% |
| Alerting (Alertmanager) | 15% |
| Grafana Integration | 5% |

### Compétences évaluées
- Comprendre la structure et les composants de Prometheus
- Configurer et gérer des exporteurs de métriques
- Utiliser PromQL pour interroger et analyser les données de métriques
- Configurer et utiliser Alertmanager pour la gestion des alertes
- Intégrer Prometheus avec Grafana pour la visualisation des données
