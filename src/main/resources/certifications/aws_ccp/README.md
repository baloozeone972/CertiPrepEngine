# AWS Certified Cloud Practitioner (CLF-C02)

## 📋 Description
La certification AWS Certified Cloud Practitioner (CLF-C02) est une certification **fondamentale** pour les professionnels souhaitant comprendre et utiliser le cloud AWS. Elle offre une introduction complète aux concepts de base du cloud, aux services principaux d'AWS et aux modèles de tarification. Cette certification est idéale pour les débutants qui souhaitent acquérir une solide base de connaissances en cloud computing avant d'explorer des certifications plus avancées.

## 🏛️ Organisme délivrant
**Amazon Web Services Inc. (AWS)** - Leader mondial du cloud computing, AWS propose une gamme complète de services pour les entreprises et les individus.

## 📅 Validité
- **Valable 18 mois**
- Renouvellement possible par examen ou certification supérieure (comme AWS Certified Solutions Architect - Associate)

## 📚 Programme détaillé
### Thèmes principaux
| Thème | Questions |
|---|---|
| Cloud Concepts | 50 |
| AWS Core Services | 100 |
| Security & Compliance | 60 |
| Billing & Pricing | 70 |
| Architecture & Design | 70 |

### Compétences évaluées
- Comprendre les concepts de base du cloud et la façon dont AWS fonctionne.
- Maîtriser les services principaux d'AWS, y compris EC2, S3, RDS et VPC.
- Comprendre les meilleures pratiques de sécurité et de conformité dans le cloud AWS.
- Maîtriser les modèles de tarification d'AWS et comment gérer le coût.
- Concevoir des architectures cloud efficaces et sécurisées.
