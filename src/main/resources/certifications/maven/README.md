## 📋 Description
La certification **Maven Professional Apache** valide la maîtrise complète de l'outil de build et de gestion de projet Java le plus utilisé en entreprise. Elle couvre la résolution de dépendances, les lifecycles, la configuration avancée des plugins, les profils environnementaux et l'architecture multi-modules.

Alignée sur les standards Apache et les évolutions récentes (Maven 3.9+, dependency mediation, reproducible builds), elle atteste d'une capacité à industrialiser, optimiser et sécuriser les chaînes de construction Java dans des contextes CI/CD modernes.

## 🏛️ Organisme délivrant
**Apache Software Foundation**. Certification alignée sur la documentation officielle `maven.apache.org` et les pratiques recommandées par la communauté Apache Maven.

## 📅 Validité
Valable **3 ans**. Renouvellement via examen de mise à jour ou passage à une certification Java architecte associée.
