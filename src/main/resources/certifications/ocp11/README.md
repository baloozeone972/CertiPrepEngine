# Oracle Certified Professional (OCP) - Java 11

## 📋 Description

La certification Oracle Certified Professional (OCP) Java 11 est une certification de niveau **intermédiaire à avancé**
qui valide les compétences approfondies en développement Java. Elle est la suite logique de l'OCA.

## 🏛️ Organisme délivrant

**Oracle University** - Division certification d'Oracle Corporation

## 📅 Validité

- **Valable à vie** (non expirable)
- Java 11 est une version LTS (Long Term Support) avec support jusqu'en 2026

## 📚 Programme détaillé

### Thèmes principaux

| Thème                             | Pourcentage |
|-----------------------------------|-------------|
| Java Modules (JPMS)               | 15%         |
| Collections et Génériques avancés | 15%         |
| Lambda et Streams API             | 20%         |
| Concurrence                       | 15%         |
| NIO.2 et I/O                      | 10%         |
| JDBC et Bases de données          | 10%         |
| Annotations et Reflection         | 5%          |
| Nouveautés Java 9-11              | 10%         |

### Compétences évaluées

- Maîtriser le système de modules (JPMS)
- Utiliser les streams parallèles
- Gérer la concurrence (ExecutorService, ForkJoin)
- Programmation fonctionnelle avancée
- Accès aux bases de données avec JDBC
- Manipulation des fichiers avec NIO.2
- Utiliser les annotations et la réflexion

## 🎯 Objectif

Devenir un développeur Java confirmé capable de concevoir des applications complexes.

## 🚀 Débouchés

- Développeur Java Senior
- Architecte technique junior
- Lead technique
- Consultant Java

## 📊 Statistiques

- **Taux de réussite** : ~60-65%
- **Temps de préparation recommandé** : 150-200 heures
- **Format examen** : QCM, 80 questions, 180 minutes
