# Oracle Certified Professional Java 17 (1Z0-829)

## 📋 Description
La certification Oracle Certified Professional Java 17 (1Z0-829) est une certification **technique** qui évalue les compétences en développement Java 17 pour les professionnels expérimentés. Cette certification couvre les dernières fonctionnalités de Java 17, offrant une approche avancée et moderne du développement Java.

Cette certification est particulièrement utile pour les développeurs souhaitant s'adapter aux dernières tendances du développement Java et acquérir une expertise en langage de programmation orienté objet, en gestion des données, et en développement concurrentiel.

## 🏛️ Organisme délivrant
**Oracle Corporation** - Leader mondial du marché des systèmes d'exploitation, des bases de données et des outils de développement.

## 📅 Validité
- **Valable 3 ans**
- Renouvellement par examen ou certification supérieure (Oracle Certified Professional, Java SE 17 Programmer)

## 📚 Programme détaillé
### Thèmes principaux
| Thème | Questions |
|---|---|
| Java Modules (JPMS) | 50 |
| Collections et Génériques | 60 |
| Lambda et Streams API | 70 |
| Concurrence | 60 |
| NIO.2 et I/O | 50 |
| JDBC et Bases de données | 50 |
| Pattern Matching (instanceof, switch) | 60 |
| Records et Sealed Classes | 50 |

### Compétences évaluées
- Maîtriser la structure et les fonctionnalités avancées de Java 17
- Comprendre et utiliser les modules Java (JPMS)
- Utiliser les collections et génériques de manière efficace
- Appliquer les concepts de Lambda et Streams API pour la programmation fonctionnelle
- Gérer la concurrence en Java
- Utiliser NIO.2 pour les opérations d'entrée/sortie
- Effectuer des opérations de base avec JDBC et les bases de données
- Utiliser le pattern matching pour l'instanceof et switch expressions
- Comprendre et utiliser les records et sealed classes
