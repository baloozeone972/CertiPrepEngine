# SonarQube Professional (SonarSource)

## 📋 Description
La certification SonarQube Professional est une certification **pratique** qui évalue les compétences en qualité et sécurité du code grâce à SonarQube. Elle permet aux professionnels de développer des applications de manière sécurisée et conforme aux normes de qualité. Cette certification est particulièrement appréciée par les développeurs, les architectes logiciels et les gestionnaires de projets.

## 🏛️ Organisme délivrant
**SonarSource** - Fournisseur mondial de solutions d'analyse de code et de qualité logicielle

## 📅 Validité
- **Valable indéfiniment**
- Renouvellement par examen ou certification supérieure

## 📚 Programme détaillé
### Thèmes principaux
| Thème | Questions |
|---|---|
| SonarQube Architecture | 40 |
| Quality Gates | 50 |
| Rules & Profiles | 50 |
| Security Analysis | 60 |
| Integration CI/CD | 50 |

### Compétences évaluées
- Comprendre et configurer la structure d'un système SonarQube
- Créer et gérer des Quality Gates pour assurer la qualité du code
- Personnaliser les règles et profils de qualité selon les besoins du projet
- Analyser des codes source pour identifier les vulnérabilités et les erreurs
- Intégrer SonarQube dans un pipeline CI/CD pour automatiser la qualité du code
