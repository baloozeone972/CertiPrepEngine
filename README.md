# CertiPrep Engine

## Application de préparation aux certifications Java

CertiPrep Engine est une application de bureau permettant de se préparer aux certifications Java (et extensible à
d'autres technologies) avec des modes d'examen réalistes, un mode libre et un mode révision.

## Fonctionnalités

- ✅ **500+ questions** pour la certification Java 21 Developer
- ✅ **Mode Examen** : conditions réelles (temps limité, notation)
- ✅ **Mode Libre** : personnalisation des thèmes, nombre de questions, durée
- ✅ **Mode Révision** : apprentissage avec correction immédiate
- ✅ **Thèmes clair/sombre** commutables
- ✅ **Internationalisation** (Français/Anglais)
- ✅ **Historique des sessions** avec SQLite
- ✅ **Export PDF** des résultats
- ✅ **Import/Export** de questions supplémentaires

## Prérequis

- Java 17 ou 21 installé
- 256 Mo RAM minimum (512 Mo recommandé)
